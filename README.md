# ExpensesTracker
Simple expenses tracker built with React

## How to run it locally
First, download or clone the repository. Then, open a terminal/cmd prompt in the folder you have the project and enter the following commands:

`npm install`

This will download and install all project dependencies. After that, enter this command:

`npm install`

This will start a local instance of the React project at `http://localhost:3000/`
